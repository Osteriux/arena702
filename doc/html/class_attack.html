Référence de la classe Attackabstract

classe abstraite qui permet de gérer une attaque.
La classe Attack est une classe abstraite qui permet de gérer les différentes attaques disponibles dans le jeu. Elle définit les propriétés communes à toutes les attaques telles que les dégâts, la vitesse, la durée de vie, et la texture. Elle fournit également des méthodes pour gérer les collisions, mettre à jour l'état de l'attaque en fonction du temps écoulé, et un constructeur pour initialiser les propriétés de l'attaque.
