Référence de la classe GameOver

gère le menu de GameOver.
La classe GameOver hérite de la classe abstraite Menu et permet de gérer le menu de fin de jeu. Elle gère les propriétés du menu telles que le titre, les options de recommencer ou quitter le jeu, et les indicateurs de sélection. Elle fournit également des méthodes pour dessiner le menu à l'écran et gérer les entrées clavier. Plus de détails...
